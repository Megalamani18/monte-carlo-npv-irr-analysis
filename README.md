# Monte Carlo Simulation — Fintech Payment & Insurance Platform for Airlines

A dynamic financial model built in Excel to evaluate investment viability 
using Monte Carlo simulation with adjustable uncertainty inputs.

## Project Inputs
- Initial Investment: £760,000
- Mean Annual Cash Flow: £194,000
- Standard Deviation: Adjustable via spinner input
- Cost of Capital: 10%
- Project Life: 10 years
- Distribution: Normal

## What the Model Does
- Simulates thousands of cash flow scenarios based on user-defined STD
- Generates dynamic NPV and IRR probability distributions
- Calculates Risk of Loss, Skewness and Kurtosis automatically
- Updates best case, worst case and base case as inputs change

## Key Metrics Tracked
- Mean & Median NPV and IRR
- Maximum and Minimum outcomes
- Risk of Loss of Value
- NPV Probability Distribution chart

## Tools Used
- Microsoft Excel
- Monte Carlo Simulation
- NPV & IRR Analysis
- Normal Probability Distribution
