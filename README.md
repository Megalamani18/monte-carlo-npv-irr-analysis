# Monte Carlo Simulation — NPV & IRR Business Proposal Analysis
A financial model built in Excel using Monte Carlo simulation to evaluate 
business proposal viability under uncertainty.

## What it covers
- Net Present Value (NPV) simulation across multiple scenarios
- Internal Rate of Return (IRR) distribution analysis
- Risk modelling using probability distributions
- Sensitivity analysis on key input variables

## Tools Used
- Microsoft Excel
- Monte Carlo simulation
- NPV & IRR financial metrics

## Key Outcomes
- Simulated 1000 scenarios to model revenue and cost uncertainty
- Identified probability of positive NPV under varying assumptions
- Evaluated project viability based on IRR vs hurdle rate
